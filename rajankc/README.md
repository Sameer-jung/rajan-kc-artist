# RAJAN KC — Immersive Landing Page

A pixel-perfect, cinematic noir landing page for Nepali rapper Rajan KC.

## Project Structure

```
rajankc/
├── index.html          ← Main entry point
├── css/
│   └── style.css       ← All styles (variables, layout, animations)
├── js/
│   └── main.js         ← Cursor, parallax, glitch, scroll reveal
└── README.md
```

## Features

- **Custom cursor** — gold dot + trailing ring, expands with contextual labels on hover
- **Hero parallax** — portrait reacts to mouse movement (X/Y transform) + scroll drift
- **Glitch effect** — chromatic aberration on the hero name, fires on random interval
- **Film grain + scanlines** — fixed atmospheric overlays via CSS SVG filter
- **Marquee ticker** — pauses on hover
- **Asymmetric track grid** — editorial CSS Grid layout, 7 tracks in non-uniform positions
- **3D card tilt** — perspective transform on track card hover
- **Scroll reveal** — IntersectionObserver stagger reveals per section
- **Platform list** — animated slide-in, directional hover fill
- **Responsive** — adapts cleanly to mobile/tablet

## How to Use

Simply open `index.html` in any modern browser. No build step required.

All media (Spotify cover art, artist photo) loads directly from Spotify CDN.
All links are real and functional.
